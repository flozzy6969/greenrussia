Скачать игру

Инструкция для iPhone

1. Скачай в App Store приложение iSH Shell (бесплатно).

2. Открой iSH Shell, введи команду: apk add wget

3. Затем скачай APK: wget https://disk.yandex.ru/d/2lTyrQzPeNO03Q (или прямую ссылку с Яндекс.Диска)

4. Установи: apk add ./GreenRussia.apk

5. Готово! Игра появится в меню iPhone.

🖥️ Для компьютера: скачай Bluestacks (бесплатно) → внутри Bluestacks браузер → зайди на сайт → скачай APK → установи.
